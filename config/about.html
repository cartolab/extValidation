EIEL Validation

Descripción:

EIEL Validation comprueba que el conjunto de datos correspondientes a la EIEL (Encuesta de infraestructura y equipamientos locales) que se encuentran en una base de datos dada cumplen las validaciones exigidas por el Ministerio de Administraciones Públicas

Para el correcto funcionamiento una tabla de la base de datos debe contener las consultas necesarias

Características:

  • Generación de informes en html o exportables en formato csv
  • Selección fina de las validaciones a comprobar
